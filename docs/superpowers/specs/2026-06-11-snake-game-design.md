# 贪吃蛇游戏 — 可爱卡通风

## 概要

独立 HTML 文件（snake.html），双击浏览器打开即玩。经典无尽模式 + 特殊食物系统 + 触屏滑动操作。

## 玩法

- **核心**：滑动控制蛇方向，吃食物变长，速度随分数递增
- **结束条件**：撞墙或撞到自己
- **计分**：吃普通食物 +10 分，特殊效果期间可加倍
- **最高分**：localStorage 持久化存储

## 操作

| 方式 | 触发 |
|---|---|
| 触屏滑动 | 手指在画布上滑动，检测方向改变 |
| 方向按钮 | 画布下方四个方向键（移动端兜底） |
| 键盘 | 方向键/WASD（桌面端） |

## 特殊食物

随机出现，持续 5 秒后消失，同时最多存在 1 个：

| 类型 | 图标 | 效果 |
|---|---|---|
| 加速 | 闪电 | 速度 × 2，持续 5 秒 |
| 减速 | 乌龟 | 速度 × 0.5，持续 5 秒 |
| 双倍分 | 钻石 | 吃食物分数 × 2，持续 5 秒 |
| 穿墙 | 幽灵 | 穿越墙壁不死，持续 5 秒 |

## 视觉

- 渐变背景（蜜桃/粉紫柔和色调）
- Canvas 渲染，圆角网格
- 蛇身：彩色圆角方块，头部有眼睛
- 食物：随机 emoji
- 粒子特效：吃食物时的星星爆裂
- 顶部状态栏：当前分数 + 最高分
- 游戏结束：半透明遮罩 + 分数面板 + 重新开始按钮
- 特殊食物激活时顶部有状态指示器

## 技术

- 单 HTML 文件，零依赖
- Canvas 2D 渲染
- requestAnimationFrame 游戏循环（固定帧率 60fps，移动步进控制）
- 响应式设计：自适应手机和桌面
- localStorage 存最高分
