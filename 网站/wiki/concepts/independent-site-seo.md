# 独立站 SEO 策略

## 独立站的特殊性

与平台（知乎/Medium）不同，独立站需要**从零建立搜索引擎信任**。没有平台的内置流量，一切依赖搜索发现。

## 关键词策略

### FAQ 矩阵
- 列出目标用户的所有长尾问题
- 每个 FAQ 一篇独立文章（500-1500 字）
- 标题 = 用户搜索词（"独立站怎么做SEO"）

### 内容库搭建
- 核心 Pillar Page（长文，全面覆盖一个主题）
- Cluster Pages（短文，每个聚焦一个细分问题）
- 互相链接形成 Topic Cluster

### 外部站点引用
- 在 GitHub/npm/Medium 等平台放置独立站链接
- 每个外部链接都是一种"投票"
- 新站优先从低竞争长尾词入手

## 技术 SEO
- 静态站点生成（SSG）保证 Google 能完整抓取
- 结构化数据（Schema.org Article/FAQ）
- Sitemap + robots.txt
- Core Web Vitals（加载速度是排名因素）
