# Vibe Coding

**来源**：Andrej Karpathy 于 2025 年提出的编程范式概念。

## 定义

Vibe Coding 是一种新型编程方式：开发者用自然语言描述想要的功能和"感觉"，AI（LLM）直接生成可运行的代码。开发者更多扮演"产品经理+QA"角色，而非逐行编写代码。

> "The hottest new programming language is English." — Andrej Karpathy

## 与 AI 代码生成器的关系

本项目中的 [[ai-code-generator]] 就是一个典型的 Vibe Coding 工具——用户输入描述，DeepSeek API 直接返回完整的 HTML 页面。

## 关键转变

| 传统编程 | Vibe Coding |
|---------|-------------|
| 写代码实现功能 | 描述需求让 AI 实现 |
| 调试语法/逻辑错误 | 审查 AI 输出是否满足需求 |
| 关注实现细节 | 关注产品体验和设计意图 |

## 延伸

与 [[llm-knowledge-base]] 一脉相承——Karpathy 认为 LLM 不仅应该写代码，更应该维护知识。
