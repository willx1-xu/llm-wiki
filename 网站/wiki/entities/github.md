# GitHub

全球最大代码托管平台，微软旗下。DA 96，高权重域名。

## GEO 价值
- GitHub 页面被 Google 快速抓取和索引
- README 内容出现在搜索结果中
- Topics/Tags 是搜索入口
- 项目活跃度影响搜索排名
- 与 npm/PyPI 等包生态互相链接，增强外链网络
