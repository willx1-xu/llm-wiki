# Google Search Console

Google 官方免费 SEO 工具，优化 Google 搜索表现的核心仪表盘。

## 核心功能
- **效果报告**：展示哪些搜索词触达了你的站点、点击率、展现量、排名
- **URL 检查**：查看某个页面是否被索引、有无抓取错误
- **Sitemap 提交**：告诉 Google 你的站点有哪些页面
- **Core Web Vitals**：页面加载性能数据
- **手动操作/安全问题**：被惩罚或黑帽标记的警告

## GEO 中的使用
- 发现意外排名的关键词 → 扩展内容
- 找到高展现低点击的页面 → 优化标题和 meta description
- 监控索引状态 → 新文章是否被抓取
