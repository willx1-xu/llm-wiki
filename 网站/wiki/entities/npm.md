# npm

JavaScript/Node.js 包管理器，全球最大开源包注册中心。

## GEO 价值
- 包名含关键词 → npm 站内搜索 + Google 收录
- package.json 的 description/keywords 直接影响可发现性
- 包的 README 被 Google 索引
- 下载量、星标、维护频率影响 npm 搜索排名
